Value SysType (Eltex ME5\d00 carrier router|Eltex ME5000M? modular carrier router)
Value SysName (\S+)
Value SysVersion (\d+\.\d+\.\d+\.\S+)
Value SysUptime (\d+ hours, \d+ minutes, \d+ seconds|\d+ days, \d+ hours, \d+ minutes|\d+ weeks, \d+ days, \d+ hours, \d+ minutes)
Value SysRestartTime (\w+ \w+  *\d+ \d+\:\d+\:\d+ \d)
Value SysMac (\w\w:\w\w:\w\w:\w\w:\w\w:\w\w)
Value SysPSM1 (not present|present, status ON|present, status OFF|Not connected|DC feeder, status ON)
Value SysPSM2 (not present|present, status ON|present, status OFF|Not connected|DC feeder, status ON)


Start
  ^  System type:            ${SysType}
  ^  System name:            ${SysName}
  ^\s+System software:\s+Eltex Network OS for ME5k version\s+${SysVersion}
  ^  System uptime:          ${SysUptime}
  ^  System restarted at:    ${SysRestartTime}
  ^  System MAC address:     ${SysMac}
  ^  Power supply 1 is    ${SysPSM1}
  ^  Power supply 2 is    ${SysPSM2}
